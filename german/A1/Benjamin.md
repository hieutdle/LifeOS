
## 12. Akkusativ

|            | m         | f        | n       | PL    | Verben                      |
| ---------- | --------- | -------- | ------- | ----- | --------------------------- |
| Nominative | der ein   | die eine | das ein | die / | sein                        |
| Akkusative | den einen | die eine | das ein | die / | haben, machen, trinken, ... |

Das ist eine Frau -> Ich sehe eine Frau.
